# Animation

PixelArt98 animation is built around the Animation window timeline. Click the Animation window first; its keyboard shortcuts only apply while that window is focused, so they do not interfere with drawing on the canvas.

## Playback

- `Space`: play or pause.
- `Enter`: stop playback.
- `Left Arrow`: previous frame.
- `Right Arrow`: next frame.
- `P`: switch between Ping-Pong and Loop playback.
- `Onion Skin`: use the checkbox in the Animation window.

## Frames

- `N`: add a blank frame after the active frame.
- `Ctrl+C` / `Cmd+C`: duplicate the active frame.
- `Ctrl+V` / `Cmd+V`: duplicate the active frame.
- `Delete` / `Backspace`: delete the active frame.
- `Shift+Left Arrow`: move the active frame left.
- `Shift+Right Arrow`: move the active frame right.
- Drag a frame clip body on the timeline to move it.
- Drag a frame clip edge to change how long that frame is held.
- Hold `Shift` while trimming for fine timing.
- Hold `Ctrl` / `Cmd` while dragging a right edge to adjacent-trim against the next frame.

## Timeline

- `Ctrl` / `Cmd` + mouse wheel: zoom the timeline.
- `Shift` + mouse wheel: scroll the timeline left or right.
- The zoom range automatically prevents frame clips from visually overlapping.
- The yellow playhead shows the active frame.
- Frame clips show their frame number and hold length.

## Cues

- `M`: create a cue at the active frame.
- Drag a cue to move it to another frame.
- Right-click a cue to rename it.
